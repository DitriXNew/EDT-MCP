# list_toolsets

List the tool groups (toolsets) used by progressive tool disclosure: each toolset's id, title, description, member tools, and whether it is currently visible in tools/list. When progressive disclosure is on, only the 'core' toolset plus toolsets you reveal with enable_toolset appear in tools/list; when off (default) all tools are already listed. Read-only. Use it to discover which toolset id to pass to enable_toolset.

## Parameters
No parameters.
